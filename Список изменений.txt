Список изменений в программе Деметра

Версия 2.5 - Добавлена возможность при использовании функции Обработка списка искать дубликаты по нескольким выбранным колонкам.
Добавлено создание файла с частотными таблицами (сколько раз то или иное значение встречается в ней) для каждой колонки при использовании функции Обработка списка.
Исправлена ошибка в функции Создание документов при сохранении по папкам в случае если в значении по которому создается название папки был пробел в начале или в конце.
Исправлена ошибка в функции Разница двух таблиц появлявшаяся в случае если различия между файлами отсутствовали.
Для всех функций добавлена возможность работы с файлами с расширением xlsm.

Версия 2.4 – Добавлена функция Выверка ЕГИССО, предназначенная для поиска дублей в файлах выверки фактов назначений по пересекающимся периодам. В функцию Подготовка данных для ЕГИССО добавлены поиск дубликатов и создание частотных таблиц по каждой колонке шаблона ЕГИССО. Добавлена проверка соответствия категории categoryID значению LMSZID. В общем файле для облегчения проверки добавлены колонки с текстовым наименованием LMSZID и categoryID.

Версия 2.3 – Добавлена функция по обработке регионального шаблона (используемый в Республике Бурятия) для ГИР ВУ.

Версия 2.2. Добавлена функция структурирования данных из таблиц (из учебных планов, рабочих программ и т.п.).
Добавлена функция поиска и исправления ошибок в файлах ЕГИССО(даты,ФИО,СНИЛС, проверка правильности заполнения связных колонок и т.п.).
Сделана необязательной кнопка выбора файла с параметрами ЕГИССО.
Исправлена ошибка когда при создании pdf файлов в названии созданного файла не писался тип документа.

Версия 2.1 - Добавлено автоматическое форматирование СНИЛС в двух форматах через дефис и без дефиса (ХХХ-ХХХ-ХХХ ХХ и ХХХХХХХХХХХ); теперь в отчете по стандарту БРИТ дополнительно считаются: Количество и список совершеннолетних  и несовершеннолетних студентов из многодетных семей, для отображения этих данных требуется наличие в эталоне и файлах данных колонки с названием Статус_Многодетная_семья с указанием значения да; Количество и список совершеннолетних  и несовершеннолетних студентов из семей участников СВО.

Версия 2.0 - Добавлена автоматическая проверка заполнения колонок с персональными данными (программа указывает в каких файлах и у каких студентов не заполнены ПД). Добавлена функция для нахождения разницы между двумя таблицами. Обновлена структура шаблона для заполнения. В новом шаблоне исправлена проблема с неработающими выпадающими списками в Excel 2007. Добавлен вариант подсчета без студентов в академическом отпуске. 

Версяи 1.9 - Деметра внесена в реестр отечественного ПО. Исправлена ошибка при склонении ФИО когда между словами стоят лишние пробелы. Исправлена проверка на соответствие эталонному файлу.

Версия 1.8 - Добавлено создание срезов по возрасту, году рождения, полу, образовательной программе для стандартного и настраиваемого отчета. Улучшена структура папок по которым раскладываются файлы.

Версия 1.7 – Добавлен новый тип обрабатываемых колонок Список_ для множественных значений. Добавлено создание сводных таблиц в разрезе возраста, группы, образовательной программы. Файлы связанные с ЕГИССО и дополнительные сводные таблицы сохраняются по папкам.

Версия 1.6 - Реализовано создание полноценных файлов для ЕГИССО, с автоматическим заполнением колонок: LMSZID,categoryID,ONMSZCode и т.д.Реализовано создание отдельных списков получателей льгот по группам. 

Версия 1.5 - В файле с персональными данными получателей льгот для ЕГИССО формируется дополнительный лист на котором указаны получатели нескольких льгот.

Версия 1.4 – Теперь на основании данных, указанных в файлах, при создании отчетов дополнительно формируется файл, содержащий в себе персональные данные получателей льгот для ЕГИССО. Проводится проверка на ошибки в данных и создается отдельный файл для таких ошибок. СНИЛС автоматически преобразовываются в формат ЕГИССО.

Версия 1.3 – Теперь при создании документов можно выбрать до трех колонок таблицы по котором будут созданы вложенные папки куда будут помещены документы. Например, если вы укажете порядковые номера колонок, где находятся Профессия, Группа ,Статус_общежитие то программа создаст соответствующую структуру папок, в которых будут находится ваши документы. 
В отчет по стандарту БРИТ  на лист Социальный паспорт добавлены колонки Обучается, Академ.
В файле Свод по статусам теперь корректно считаются студенты, у которых не проставлен статус в колонке Статус_Учёба.
Исправлена ошибка из за которой возникали проблемы при работе в 32 битной версии Windows.

Версия 1.2 – Добавлено создание колонок с возрастными категориями (Текущпй возраст,Совершенолетие, Порядковый_номер_месяца_рождения, Название_месяца_рождения, Год_рождения, Один_ПК, Один_ПО, СПО_Один, Росстат) 
Добавлено создание файла с аналитикой по колонкам с возрастными категориями.
Можно выбирать дату, на которою нужно подсчитать аналитику по возрастам.
